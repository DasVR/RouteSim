# Support

RouteSim is a community fork focused on route-based location simulation for developer testing.

- **Bug reports:** [GitHub Issues](https://github.com/DasVR/RouteSim/issues/new/choose)
- **Feature requests:** use the Feature request template
- **Upstream tunnel / pairing help:** [StikDebug Guide — pairing file](https://github.com/StephenDev0/StikDebug-Guide/blob/main/pairing_file.md)

RouteSim does not collect personal data. Pairing files and saved routes remain on your device.
